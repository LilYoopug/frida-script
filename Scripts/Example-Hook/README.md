# Example Hook Script

## Description

This is an example Frida hook script for Android applications.

## Features

- Hook Java methods
- Log function calls and parameters
- Intercept and modify return values

## Usage

1. Load this script in JsHook
2. Select target application
3. Enable the script
4. Monitor the logs for hooked function calls

## Compatibility

- Android 5.0+
- Frida 16.0+
